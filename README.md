# project

Standard repository skeleton for new and ongoing GitHub projects.

## Structure

- `prompts/`: reusable prompting assets
- `data/`: raw to processed data lifecycle
- `sql/`: source notes, query scripts, and exports
- `notebooks/`: exploratory and analytical notebooks
- `src/`: production code
- `reports/`: draft and final reporting outputs
- `evidence/`: screenshots, exports, and execution logs
- `checks/`: validation and QA checklists
- `.vscode/`: team editor settings and recommended extensions
